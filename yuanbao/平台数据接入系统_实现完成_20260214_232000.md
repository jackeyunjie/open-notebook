## 支持平台

**国内：**
小红书、微博、公众号、知乎、B站、抖音、快手

**海外：**
Twitter/X、Instagram、YouTube、LinkedIn

---

## 使用方式

1. IP中心 → 平台管理 标签 → 点击"管理平台账号"
2. 选择平台（如小红书）
3. 粘贴浏览器Cookie（从开发者工具复制）
4. 系统自动同步内容到P0感知层

---

## 数据流向

```
平台API → Connector → 标准化数据 → 数据库 → P0感知 → P1-P2-P3
```
