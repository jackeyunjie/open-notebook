
# 🤖 一人公司/超级个体 AI创作Prompt合集

## 一、选题生成Prompt

### 1.1 批量选题生成
```
你是一位资深的[一人公司/自由职业]领域内容创作者，目标受众是[想要转型自由职业的职场人]。

请为我生成50个[小红书]的选题方向，要求：
1. 覆盖入门、进阶、高阶不同阶段
2. 包含方法论、案例、工具、故事不同类型
3. 有爆款潜力，能引发共鸣或提供价值
4. 符合平台调性
5. 每个选题标注适合的优先级（高/中/低）

请按以下类别整理输出：
- 入门认知类（10个）
- 技能变现类（10个）
- 个人品牌类（10个）
- 效率工具类（10个）
- 心态成长类（10个）
```

### 1.2 热点选题挖掘
```
请分析当前[自由职业/一人公司]领域的热门话题和趋势。

基于以下维度，生成10个借势热点选题：
1. 行业趋势热点
2. 社会情绪热点
3. 季节性/节日热点
4. 平台活动热点

每个选题需要包含：
- 选题标题
- 热点关联点
- 切入角度
- 预估互动率
```

## 二、内容创作Prompt

### 2.1 小红书图文创作
```
请为选题"[什么是一人公司？]"创作一篇小红书图文内容。

要求：
1. 标题要吸引人，使用数字或悬念
2. 开头3秒抓住注意力
3. 正文用 bullet points 呈现，每点配emoji
4. 内容结构：痛点引入 → 核心内容（3-5点）→ 行动号召
5. 总字数控制在800字以内
6. 添加5-8个相关话题标签
7. 结尾引导互动（提问或征集）

输出格式：
标题：
正文：
标签：
```

### 2.2 公众号深度文创作
```
请为选题"[一人公司商业模式详解]"创作一篇公众号深度文章。

要求：
1. 标题要有价值感和传播性
2. 开头用故事/数据/问题引入，建立共鸣
3. 正文结构清晰，分章节论述
4. 包含具体案例和数据支撑
5. 有方法论总结，可操作性强
6. 目标字数：2000-3000字
7. 结尾有金句总结和行动号召

文章大纲：
- 什么是一人公司
- 一人公司的5种商业模式
- 每种模式的优缺点分析
- 如何选择适合自己的模式
- 从0开始的启动建议

输出完整文章。
```

### 2.3 抖音口播脚本创作
```
请为选题"[如何判断自己是否适合一人公司]"创作一个抖音口播脚本。

要求：
1. 时长控制在60秒以内
2. 开头3秒必须有钩子
3. 内容结构：钩子 → 问题 → 3个判断标准 → 行动建议
4. 语言口语化，适合口播
5. 每句话标注语气（平静/强调/疑问等）
6. 标注需要配合的画面/表情

输出格式：
【标题】
【时长】
【脚本】
  0-3秒：...
  3-15秒：...
  ...
【画面建议】
【BGM建议】
```

### 2.4 知乎回答创作
```
请回答知乎问题："[自由职业真的比上班好吗？]"

要求：
1. 回答要有专业度和深度
2. 客观分析优缺点，不偏激
3. 结合自身经验或真实案例
4. 结构清晰，分点论述
5. 目标字数：1000-1500字
6. 结尾引导点赞关注

回答框架：
- 直接回答核心问题
- 自由职业的优点（3点）
- 自由职业的缺点（3点）
- 适合什么样的人群
- 个人建议和总结
```

```

要求：
1. 每篇提取1个核心观点深度展开
2. 每篇字数500-800字
3. 使用小红书风格（emoji、bullet points）
4. 每篇配不同的标题和标签
5. 3篇内容要有连贯性，可以做成系列

输出3篇完整的小红书图文。
```

### 3.2 长文转抖音脚本
```
请将以下文章改写成3个抖音口播脚本：

要求：

输出3个完整脚本，包含标题、时长、正文。
```

## 💡 使用建议

1. **替换方括号内容**：将所有`[...]`替换为你的具体信息
2. **多次迭代**：不满意可以让AI重新生成或调整
3. **人工润色**：AI生成后务必加入个人风格和真实案例
4. **测试验证**：不同Prompt效果不同，找到最适合你的

---

> 记住：AI是助手，不是替代。你的独特经历和观点才是最宝贵的。
