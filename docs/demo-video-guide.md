# Open Notebook Demo Video - Recording Guide

## üì± Using the HTML Showcase (Recommended)

The `skills-showcase.html` is a self-contained animated presentation that you can record for social media.

### How to Record

#### Option 1: Browser Screen Recording

1. **Open the HTML file:**
   ```bash
   # Use any browser
   open docs/demo-video-assets/skills-showcase.html
   # or
   start docs/demo-video-assets/skills-showcase.html
   ```

2. **Set browser to mobile viewport:**
   - Chrome/Edge: F12 ‚Üí Toggle Device Toolbar (Ctrl+Shift+M)
   - Select "iPhone 12 Pro" (390x844) or "Pixel 5" (393x851)

3. **Start screen recording:**
   - Windows: Xbox Game Bar (Win+G) or OBS
   - Mac: QuickTime Player ‚Üí File ‚Üí New Screen Recording
   - Use OBS for best quality

4. **Let it auto-play:**
   - Scene 1: 3s (Hook)
   - Scene 2: 8s (10 Skills)
   - Scene 3: 5s (Comparison)
   - Scene 4: 4s (Privacy)
   - Scene 5: 5s (CTA)
   - Total: ~25 seconds

#### Option 2: OBS Studio Setup

```
Settings:
- Resolution: 1080x1920 (9:16 for TikTok/Reels)
- FPS: 30
- Output: H.264, 10 Mbps

Scene Setup:
1. Add "Browser Source"
2. URL: file:///path/to/skills-showcase.html
3. Width: 1080, Height: 1920
4. Record directly
```
