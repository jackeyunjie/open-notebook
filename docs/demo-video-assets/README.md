## 🚀 快速开始（中文版）

### 方法1：直接录制HTML页面（最简单）

```bash
# 1. 打开中文版展示页面
open skills-showcase-zh.html

# 2. 浏览器F12 → 手机模式
# 选择 iPhone 12 Pro (390×844)

# 3. 开始屏幕录制

# 4. 页面会自动播放5个场景：
#    - 开场钩子（4秒）
#    - 10技能展示（10秒）
#    - 对比表（6秒）
#    - 隐私强调（5秒）
#    - 行动号召（6秒）
```

### 方法2：配合语音解说（专业版）

1. **准备语音**：
   - 阅读 `voiceover-zh.md` 中的30秒或60秒脚本
   - 用手机或录音设备录制
   - 保存为 `voiceover.mp3`

2. **视频编辑**（使用剪映/PR/达芬奇）：
   - 导入录制的HTML视频
   - 导入语音文件
   - 对齐语音与画面
   - 添加字幕（导入SRT文件）

3. **导出**：
   - 抖音/视频号：1080×1920（9:16）
   - B站/YouTube：1920×1080（16:9）

## 🎬 制作检查清单

### 录制前
- [ ] 浏览器缩放比例125%（确保文字清晰）
- [ ] 开启深色模式（更酷的视觉效果）
- [ ] 关闭所有通知
- [ ] 清理浏览器书签栏

### 录音时
- [ ] 环境安静（无空调/风扇噪音）
- [ ] 语速适中，吐字清晰
- [ ] 情绪饱满，有感染力
- [ ] 每个版本录3遍，选最佳

### 后期制作
- [ ] 视频与语音对齐
- [ ] 字幕导入SRT文件
- [ ] 添加背景音乐（音量-20dB）
- [ ] 添加转场效果
- [ ] 首尾添加Logo

### 导出设置
- [ ] 分辨率：1080×1920（竖屏）或1920×1080（横屏）
- [ ] 帧率：30fps
- [ ] 码率：8-10Mbps
- [ ] 格式：MP4（H.264）

### 发布前
- [ ] 预览检查字幕是否清晰
- [ ] 确认音频无爆音/杂音
- [ ] 检查第一帧是否吸引人
- [ ] 确认链接可点击
- [ ] 选择最佳发布时间（晚8-10点）

## 🆘 常见问题

### Q: 需要专业录音设备吗？
A: 不需要。手机耳机麦克风或电脑内置麦克风即可，关键是环境安静。

### Q: 可以用剪映一键生成字幕吗？
A: 可以。剪映的自动识别准确率较高，也可以直接导入提供的SRT文件。

### Q: 竖屏和横屏都要做吗？
A: 建议优先做竖屏（抖音/视频号流量大），B站可以再做一个横屏版。

### Q: 英文版怎么做？
A: 使用 `skills-showcase.html`（英文版），配音参考 `voiceover-zh.md` 中的英文部分。
