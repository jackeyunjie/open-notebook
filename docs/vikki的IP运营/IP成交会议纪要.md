- IP 底层逻辑与选题认知
  - IP 本质与价值观同步
    - IP 本质定义：Vikki 提出 IP 本质是基于人的共识成交，成交是最终目的，共识是从陌生人到付费的关键环节，人是个人 IP 的关键要素。
    - 价值观同步：强调做 IP 要同步价值观，包括 IP 终局是影响力、想都是问题做都是答案、站在岸上学不会游泳等观点，鼓励大家在分享中互动。
  - 选题与内容的区别和重要性
    - 概念区分：通过互动探讨，明确选题是成交的先决条件，决定成交上限，要考虑写给谁、写什么、负责哪一步；内容决定转化效率，要满足用户期待，推进用户决策。
    - 常见问题分析：调研发现大家在选题和内容上存在方向模糊、缺乏吸引力、与用户需求不符、无法持续、有流量无转化等问题。
- 破除选题误区
  - 泛选题与成交的关系
    - 垂直与泛选题定义：以高中物理提分老师为例，解释垂直选题与产品强相关，泛选题与产品弱相关甚至不相关。
    - 泛选题价值：Vikki 认为泛选题不等于不成交，离用户最近的是价值观，通过泛选题先共情再带路，吸引用户，扩大用户圈子，实现品牌加获客双丰收。
  - 泛人群与成交的关系
    - 垂直与泛人群定义：以房地产销售为例，说明垂直人群是当下有需求的用户，泛人群包括潜在用户，如即将结婚、生小孩未来可能买房的人。
    - 挖掘潜在用户：强调不能只关注已产生需求的用户，要挖掘潜在用户，通过满足他们前端决策的需求，激发其未来的购买需求。
- 选题四象限地图构建
  - 四象限划分依据
    - 横轴与纵轴定义：横轴为选题与产品的关系，分为垂直选题（与产品强相关）和泛选题（与产品弱相关）；纵轴为人群的清晰度，分为垂直人群（当前精准用户）和泛人群（目前泛观众）。
    - 四象限形成：两轴交叉形成四个象限，分别对应不同类型的内容和用户。
  - 四象限内容解读
    - 象限定位与用户：明确四个象限分别为意向用户（你有病我有药）、意识用户（你有病我懂你）、大众用户（让更多人看见我）、潜在用户（让更多人看见药），并阐述每个象限对应的用户特征。
    - 目的与价值：每个象限有明确目的，如推进决策、建立偏好、扩大认知、唤醒需求；交付不同价值，如解题价值、共鸣立场、引流价值、场景唤醒等。
    - 选题类型：对应不同象限，选题类型包括痛点选题、情绪选题、热点选题、认知选题。
- 各象限选题实操方法
  - 痛点选题（意向用户）
    - 痛点类型：分为即时性痛点（如手机内存清理、人力约谈赔偿、演唱会抢票等）和持续性痛点（如装修预算超支、不会做选题、换季皮肤干燥等）以及隐性痛点（如妈生整容、用 AI 创作不想被发现、给宝宝买辅食被婆婆说不称职等）。
    - 选题方法：即时性痛点可通过搜索平台获取；持续性痛点可记录用户问烦的问题；隐性痛点需到用户放松安全的环境洞察，如蒙氏教育专家通过潜伏妈妈群挖掘痛点，使账号流量增长。
  - 情绪选题（意识用户）
    - 用户情绪来源：用户情绪主要包括怕（怕选错、来不及、白忙活等）、委屈（被误解、否定、拖累等）、羞耻（觉得不配、不好意思等）、愤怒（被割韭菜、忽悠、行业乱等）。
    - 选题生成方法：先收集用户原话，再归类到四种情绪，最后用 “你有病，我懂你” 句式生成选题，如怕的情绪可表述为 “你不是犹豫，是害怕什么的代价你承受不起”。
  - 热点选题（大众用户）
    - 热点类型：分为长期热点（如人性底层需求，赚钱、健康、被爱等）、中期热点（周期性火爆，如开学毕业、行业招聘季等）、短期热点（突然发生，如明星事件）。
    - 选题方法：采用大话题写小的方式，找角度切入热点，如马伊琍恋爱事件可从亲子、年龄、IP、传播行业等角度解读；有效热点需考虑流量爆发力、人设契合度、长尾生命力。
  - 认知选题（潜在用户）：通过场景唤醒（如卖书时让用户找到代入感）和结果唤醒（如招商时让用户看到赚钱结果）激发用户需求。
- 选题配比与内容商业性
  - 选题配比建议
    - 不同阶段选题侧重：起号期和破圈期多关注大众用户和意识用户选题，稳定获客期侧重潜在用户和意向用户选题，冲转化期集中解决意向用户痛点。
    - 具体情况具体分析：无法给出 100% 配比答案，需根据用户反馈和具体情况调整。
  - 内容商业性解读
    - 提前完成交易决策：内容的商业性在于让用户在内容里提前完成交易条件决策，降低最终谈判成本，很多成交用户在内容中已完成大部分决策。
    - 成交是体系问题：强调成交不是技巧问题，是体系问题，创作者要建立选题和内容体系。
- 答疑环节
  - 模型适用性问题：Vikki 表示选题四象限模型适用于绝大部分内容创作，包括公众号，如在公众号总结的 part 模型中也包含相关选题思路。
  - 内容区隔问题：对于成交用户内容和实际付费课程内容，知识型或专家型创作者可采用顾问式成交，先解决小问题，再逐步解决大问题，公开内容秀肌肉，付费内容给价值。
  - 平台运营问题：建议根据用户所在平台分配精力，以用户多的平台为主运营平台，其他平台做分发；部分平台会给账号打标签，如小红书标签影响较大，可根据情况战略性放弃或利用。
  - 直播区别问题：公域直播面对陌生用户，有违禁词限制，内容用于筛选用户；私域直播面对有兴趣的用户，更注重说服用户。
  - 产品象限问题：ToB to g 产品第一象限讲安全合规、价格合理的解决方案；第二象限从情绪角度介入，了解核心问题；第三象限结合业务找热点；第四象限激发新产品解决问题的场景。
